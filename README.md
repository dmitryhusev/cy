# Cypress project


## Prerequisites
- [Node.js](https://nodejs.org)
- [npm](https://www.npmjs.com/get-npm)

## Usage

1. Open project in the editor/ide
2. In the root of the project run
```bash
npm install
```
3. Launch application
```bash
node app.js
```
4. Open another shell and run the tests (which are ./cypress/e2e)
```
npx cypress run
```
